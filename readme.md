🏆 Défi du Jour 2 : Le Jeu du Juste Prix
Créez une variable prix_secret contenant le nombre 42.
Demandez à l'utilisateur de deviner le prix en utilisant input("Devinez le prix : ") (convertissez sa réponse en entier avec int()).
Tant que l'utilisateur n'a pas trouvé le bon prix :
Si son nombre est plus petit, affichez "C'est plus !".
Si son nombre est plus grand, affichez "C'est moins !".
Redemandez-lui un nombre.
S'il trouve, affichez "Bravo, vous avez gagné !"
